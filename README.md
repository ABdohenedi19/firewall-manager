# Firewall Manager

**Firewall Manager** is a Python-based tool to manage your computer's firewall efficiently. It allows you to control firewall rules, block or allow ports and protocols, monitor activity in real-time, and log events for later analysis.

---

## Features
- Add and remove firewall rules easily.
- Block or allow specific ports and protocols.
- Real-time monitoring of firewall activity.
- Log all activities to a file for analysis.
- Simple API for extending the tool with custom scripts.

---

## Requirements
- Python 3.12 or higher
- Python libraries:
  - `requests`
  - `rich`
  - Any other dependencies included in `Lib/site-packages/`
